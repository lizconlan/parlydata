parlydata.com

Hello, you are currently viewing a pre-beta version of "parlydata" - an exploration of historical data about the UK Parliament. This is currently a very limited pilot with just a few datasets, mostly centred around the last 5 General Elections and isn't being updated to reflect current events (so if, in a few weeks, it still shows Louise Mensch as your local MP, please don't write in and complain). The idea is to present this information in the form of a RESTful API and see if it looks like something which could be genuinely useful if more data was poured into it.

Although the code is open source and public, I am not yet in a position to ask for help as the software constitutes the "practical" component of my Open University degree project so has to be my own work through the current phase and all my efforts are currently focussed on writing that up to get as good a mark as I can. After that, if it looks useful... Well, we'll see.

Please note - this version of the API is the one I am submitting to the Open University for assessment, as such it is a frozen copy which will not be updated.

Acknowledgements

The main data sources for this pilot were:

  • The UK Parliament authoritative source of all things Parliament-related
  • The BBC without whose archives, accurate election results would have proved impossible to recover

  • Historic Hansard
  • Wikipedia

My thanks for more indirect data assistance to "Who's Who", chiefly for settling disputes over matters of spelling and birth dates of noted parliamentarians, ScraperWiki for making the data mining as painless as possible, and the work of Leigh Rayment.
